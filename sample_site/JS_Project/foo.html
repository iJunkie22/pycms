4 - Positioning And Layout With CSS

Positioning

  • Static css position: static;
  • Relative CSS:
    • Absolute css position: absolute;

  • Fixed This stays in place regardless of scrolling

    position: fixed;

    Other CSS traits

    These specify the offsets:

    element{
    left|right|top|bottom: 25px;
    float: left|right;
    clear: left|right|both;
    }

This makes the float reserve the specified zone instead of sharing.

Layering

z-index: [NUM]; This specifies which layer it should be sorted as
